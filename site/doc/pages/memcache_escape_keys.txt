:sig
StringList memcache_escape_keys(StringList keys)

:params
keys : list of application keys
return value : escaped keys

:see
>sys
>memcache_escape_key

:content
Applies `memcache_escape_key()` to each key in a list.

:example
StringList keys;
keys.push_back("a b");
keys.push_back("c\td");
print(join(memcache_escape_keys(keys), ", "), "\n");
